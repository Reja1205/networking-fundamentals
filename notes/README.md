# Cisco Packet Tracer Labs

This repository contains my Cisco Packet Tracer labs, notes, and networking practice activities.

## What I Learned

### Cisco Packet Tracer

Cisco Packet Tracer is a network simulation tool that allows me to build and test networks without physical networking equipment.

### Basic Networking Concepts

#### IP Address

An IP address is a unique address used to identify a device on a network.

#### Router

A router connects different networks and uses IP addresses to send data to the correct destination.

#### Switch

A switch connects devices within the same local network (LAN).

#### Packet

A packet is a small unit of data that travels across a network. It contains source and destination information.

#### Ping

Ping is a tool used to test connectivity between network devices.

### Network Configuration

Network configuration means setting up network devices so they can communicate with each other.

Examples:

* Assigning IP addresses
* Connecting devices
* Configuring routers and switches

### Network Troubleshooting

Network troubleshooting means finding and fixing network problems.

Examples:

* Incorrect IP address
* Connectivity issues
* Configuration errors

### Network Topology

Network topology is the layout of a network and shows how devices are connected.

Example:

PC → Switch → Router → Internet

### Packet Tracer File Types

#### .pkt

A Packet Tracer project file used to save network topologies and configurations.

#### .pka

A Packet Tracer activity file that includes instructions, scoring, and assessments.
